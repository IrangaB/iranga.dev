# iranga.dev

Personal portfolio for **Iranga Basnayake** — Senior System Analyst & AI Engineer.

🌐 Live at **[iranga.dev](https://iranga.dev)**

## Stack

- Static HTML / CSS / vanilla JS — no frameworks, no build step
- [Manrope](https://fonts.google.com/specimen/Manrope) (display) + [Inter](https://fonts.google.com/specimen/Inter) (body) + [JetBrains Mono](https://www.jetbrains.com/lp/mono/) (code/labels)
- Hosted on [GitHub Pages](https://pages.github.com/) behind [Cloudflare](https://www.cloudflare.com/)
- Form backend: [Web3Forms](https://web3forms.com/)
- Analytics: [Cloudflare Web Analytics](https://www.cloudflare.com/web-analytics/) (privacy-first, no cookies)

## Local development

```bash
# Serve locally
python3 -m http.server 5050
# Open http://localhost:5050
```

## Deploying

Push to `main` → GitHub Actions auto-deploys to GitHub Pages → live at `iranga.dev` within ~60 seconds.
