ILAB — the digital lab for textile quality control.

ILAB (Matrix Digital Lab) replaces paper-based quality testing with a digital system where every test, every sample, and every result is traceable, benchmarked against buyer specifications, and instantly auditable. Built end-to-end by Iranga Basnayake.

The problem

Textile quality labs run dozens of tests per fabric — colour fastness, tensile strength, shrinkage, pilling resistance — each tested against specifications from each buyer. Done on paper, this generates thousands of pages per month with no easy way to find what failed, why, or whether the same defect appeared three months ago.

The solution

ILAB is a single platform where every test is captured digitally, with photos, machine readings, and operator notes — and reconciled against the buyer’s spec sheet automatically.

Capabilities

  • Buyer spec library — every buyer’s acceptance criteria stored once and reused per order.
  • Test workflows — guided test execution with mandatory photo evidence and machine readings.
  • Auto pass/fail — tested values compared against the buyer’s spec; pass/fail decided by software, not memory.
  • Defect history — search every defect, every fabric, every cycle, going back years.
  • Buyer-ready reports — auto-generated PDF reports per shipment, signed digitally.

Stack

Django PostgreSQL React React Native (mobile) PDF generation

My role

Architecture, backend, web and mobile frontends, and rollout — including training the lab team and integrating with existing buyer reporting workflows.
